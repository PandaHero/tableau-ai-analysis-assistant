# 前端界面视觉设计图（简洁版 - Tableau 风格）

## 设计理念

采用 Tableau 设计语言，保持界面简洁专业，专注于核心对话功能。

## 设计参考

本设计基于 `.kiro/specs/frontend-redesign/` 的 Tableau 风格设计，采用：
- Tableau 10 色板（主蓝色 #1F77B4）
- 三区域布局（HeaderBar 48px + ChatContainer flex + InputArea 64px）
- 两页结构（首页 → 对话页面）

## 1. 三区域布局结构

```

│                        HeaderBar (48px 固定)                             │

│                                                                          │
│                     ChatContainer (flex: 1, 可滚动)                      │
│                                                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                        InputArea (64px 固定)                             │
└─────────────────────────────────────────────────────────────────────────┘
```

**布局规范**：

| 区域 | 高度 | 说明 |
|-----|------|------|
| HeaderBar | 48px | 固定高度，白色背景，底部边框 |
| ChatContainer | flex: 1 | 占据剩余空间，浅灰背景 #FAFAFA，可滚动 |
| InputArea | 64px | 固定高度，白色背景，顶部边框 |

## 8. 布局规范详细说明

### 间距系统（8px 基准）

| 间距 | 值 | 用途 |
|-----|-----|------|

| md | 16px | 标准间距（消息间距、区块内边距） |
| lg | 24px | 大间距（区块间距） |

| 圆角 | 值 | 用途 |
|-----|-----|------|

所有颜色属性同时过渡

## 11. 设计规范总结

### 核心原则

1. **Tableau 风格**：采用 Tableau 10 色板和设计语言
2. **简洁至上**：移除不必要的元素，保持界面清爽
3. **视觉层次**：通过大小、颜色、间距建立清晰的层次
4. **一致性**：统一的配色、间距、圆角、阴影系统
5. **响应式**：适配桌面、平板、移动端（320px - 1920px）
6. **可访问性**：确保对比度、触摸目标、键盘导航

### 关键改进点

✅ **采用 Tableau 设计语言**：使用 Tableau 10 色板，主蓝色 #1F77B4
✅ **三区域布局**：HeaderBar (48px) + ChatContainer (flex) + InputArea (64px)
✅ **两页结构**：首页（欢迎页）→ 对话页面
✅ **移除侧边栏**：全屏对话，最大化内容空间
✅ **设置面板右侧滑出**：360px 宽度，覆盖在内容上方
✅ **优化配色**：使用 Tableau 专业配色方案
✅ **改进布局**：更合理的间距和圆角
✅ **增强动画**：流畅的过渡效果

---

这就是优化后的简洁版界面设计，采用 Tableau 设计语言，专注于核心对话功能！
