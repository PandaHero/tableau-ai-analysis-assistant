## 注意事项

1. **遵循 Tableau 设计语言**: 所有组件必须使用 Tableau 10 色板
2. **性能优先**: 确保设置页面不卡顿，使用懒加载和虚拟滚动
3. **响应式设计**: 支持 320px - 1920px 所有屏幕尺寸
4. **可访问性**: 符合 WCAG 2.1 AA 标准
5. **代码质量**: 使用 TypeScript，遵循 Vue 3 最佳实践
6. **测试覆盖**: 单元测试覆盖率 ≥ 80%
