# 前端 UI 视觉升级实施方案

根据 `.kiro/specs/frontend-redesign/ui-design` 设计文档，本方案旨在全面提升界面的视觉美感和交互体验。

## 1. 核心视觉系统升级 (Design System)

首先统一全局样式变量，建立专业的视觉基础。

*   **色彩系统**：严格遵循 Tableau 10 色板。
    *   主色调：`#1F77B4` (Tableau Blue)
    *   背景色：应用背景 `#FAFAFA` (浅灰)，内容卡片 `#FFFFFF` (纯白)
    *   边框色：`#E0E0E0`
*   **排版与间距**：
    *   基础间距：`4px` (xs), `8px` (sm), `12px` (md), `16px` (lg)
    *   圆角系统：小组件 `8px`，大卡片/气泡 `12px`
*   **阴影深度**：增加层次感，避免界面扁平化。
    *   卡片阴影：`0 2px 8px rgba(0, 0, 0, 0.08)`

## 2. 关键组件重构计划

### 2.1 整体布局 (Layout)
*   **目标**：消除"空旷感"，建立清晰的视觉分区。
*   **变更**：
    *   `ChatContainer` 背景设为 `#FAFAFA`，营造沉浸式对话区。
    *   `HeaderBar` 和 `InputArea` 保持纯白背景，添加 `1px solid #E0E0E0` 分割线。
    *   **阴影增强**：Header 增加微弱阴影 `box-shadow: 0 1px 2px rgba(0,0,0,0.05)` 使其浮起。

### 2.2 输入区域 (Input Area) - *视觉重点*
*   **现状问题**：输入框过大，缺乏精致感，像一个粗糙的文本域。
*   **设计方案** (参考 `input.md`)：
    *   **悬浮感设计**：输入框不再占满整个底部，而是一个居中的、带圆角 (`8px`) 和边框的容器。
    *   **发送按钮**：改为独立的 **40x40px 圆形按钮**，悬浮在输入框右侧或内部右侧，主色背景 (`#1F77B4`)，白色图标。
    *   **交互细节**：输入框获得焦点时，边框变为主色，并添加光晕效果。
    *   **字符计数**：右下角添加微小的字符计数器。

### 2.3 欢迎首页 (Welcome Page) - *第一印象*
*   **现状问题**：单一的大白卡片，排版生硬。
*   **设计方案**：
    *   **Hero 区域**：顶部展示大号 Logo 和 "Tableau AI 助手" 标题，配以友好的副标题。
    *   **建议卡片网格**：将"试一试"的问题拆分为 **3个独立的卡片**，横向排列（移动端纵向）。
        *   每个卡片带有轻微的 hover 上浮效果和边框变色。
        *   卡片内包含图标（如 📈, 🌍, 📊）增加视觉吸引力。

### 2.4 消息气泡 (Chat Bubbles)
*   **用户消息**：
    *   背景色：`#1F77B4` (主色)
    *   文字色：`#FFFFFF`
    *   形状：圆角 `12px`，右上角（或右下角）微调圆角以指示方向。
*   **AI 消息**：
    *   背景色：`#FFFFFF`
    *   边框：`1px solid #E0E0E0`
    *   **思维链展示**：将 "思考过程" (Thinking) 设计为可折叠的灰色区块，与最终回答区分开。

### 2.5 数据展示 (Data Display)
*   **表格优化**：表头浅灰背景 `#F5F5F5`，单元格增加内边距，避免拥挤。
*   **洞察卡片**：使用左侧色条 (Border-left) 区分 "发现"(蓝)、"异常"(橙)、"建议"(绿)，而不是简单的文本列表。

## 3. 执行步骤

1.  **基础样式**：更新 `variables.scss`，定义新的颜色和阴影变量。
2.  **布局调整**：修改 `ThreeZoneLayout.vue` 背景色。
3.  **Header 优化**：微调 `HeaderBar.vue` 样式和阴影。
4.  **输入框重写**：重写 `InputArea.vue`，实现悬浮胶囊样式。
5.  **首页重设计**：重写 `WelcomePage.vue`，实现 Hero + Grid 布局。
6.  **消息样式**：更新 `MessageItem.vue` 和 `ThinkingIndicator.vue`。
