# 输入区域设计

## 基本结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        InputArea (64px)                                  │
│                                                                          │
│  ┌─────────────────────────────────────────────────────────────┐ ┌───┐ │
│  │ 请输入您的问题...                                            │ │ ➤ │ │
│  │                                                              │ │   │ │
│  └─────────────────────────────────────────────────────────────┘ └───┘ │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## 输入框规格

| 属性 | 值 | 说明 |
|-----|-----|------|
| 最小高度 | 40px | min-height |
| 最大高度 | 120px | max-height |
| 扩展方式 | 自动 | 根据内容自动扩展 |
| 背景色 | #FFFFFF | 白色 |
| 边框 | 1px solid #E0E0E0 | 默认边框 |
| 圆角 | 8px | border-radius |
| 内边距 | 12px 16px | padding |
| 字号 | 14px | font-size |
| 占位符色 | #999999 | placeholder color |

## 发送按钮规格

| 属性 | 值 | 说明 |
|-----|-----|------|
| 尺寸 | 40x40px | 正方形 |
| 圆角 | 8px | border-radius |
| 背景色 | #1F77B4 | Tableau 蓝 |
| 图标色 | #FFFFFF | 白色 |
| 图标 | ➤ 或 发送图标 | - |

## 状态设计

### 默认状态

```
┌─────────────────────────────────────────────────────────────┐ ┌───┐
│ 请输入您的问题...                                            │ │ ➤ │
│                                                              │ │   │
└─────────────────────────────────────────────────────────────┘ └───┘
                                                                  ↑
                                                            opacity: 0.5
                                                            cursor: not-allowed
```

- 输入框：默认边框 #E0E0E0
- 发送按钮：禁用状态（opacity: 0.5, cursor: not-allowed）

### 获得焦点

```
┌─────────────────────────────────────────────────────────────┐ ┌───┐
│ |                                                            │ │ ➤ │
│                                                              │ │   │
└─────────────────────────────────────────────────────────────┘ └───┘
  ↑
  border: 2px solid #1F77B4
```

- 输入框：蓝色边框 2px #1F77B4
- 光标闪烁

### 有内容

```
┌─────────────────────────────────────────────────────────────┐ ┌───┐
│ 各产品线的销售额是多少？                                     │ │ ➤ │
│                                                              │ │   │
└─────────────────────────────────────────────────────────────┘ └───┘
                                                                  ↑
                                                            opacity: 1
                                                            cursor: pointer
```

- 发送按钮：启用状态（opacity: 1, cursor: pointer）

### 多行输入

```
┌─────────────────────────────────────────────────────────────┐ ┌───┐
│ 请帮我分析以下几个问题：                                     │ │ ➤ │
│ 1. 各产品线的销售额                                          │ │   │
│ 2. 各区域的销售趋势                                          │ │   │
│ 3. 同比增长情况                                              │ │   │
└─────────────────────────────────────────────────────────────┘ └───┘
  ↑
  高度自动扩展（最大 120px）
```

### AI 处理中

```
┌─────────────────────────────────────────────────────────────┐ ┌───┐
│ AI 正在思考...                                               │ │ ⏳ │
│                                                              │ │   │
└─────────────────────────────────────────────────────────────┘ └───┘
  ↑                                                               ↑
  disabled                                                    disabled
  opacity: 0.7                                                opacity: 0.5
```

- 输入框：禁用，显示"AI 正在思考..."占位符
- 发送按钮：禁用，显示加载图标

### 字符超限

```
┌─────────────────────────────────────────────────────────────┐ ┌───┐
│ 很长的文本内容...                                            │ │ ➤ │
│                                                              │ │   │
│                                              1987/2000 ⚠️    │ │   │
└─────────────────────────────────────────────────────────────┘ └───┘
```

- 显示字符计数器
- 接近限制时（>1800）显示警告色
- 达到限制（2000）时阻止继续输入

## 键盘交互

| 按键 | 行为 |
|-----|------|
| Enter | 发送消息（如果有有效内容） |
| Shift + Enter | 插入换行符 |
| Escape | 清空输入框（可选） |

## 输入验证

### 空白字符串

```
输入: "   \n\t  "  →  发送按钮禁用，不发送
```

- 仅包含空格、换行、制表符的字符串视为无效
- 发送按钮保持禁用状态

### 字符限制

```
输入超过 2000 字符  →  阻止继续输入，显示计数器
```

## CSS 实现

```css
/* 输入区域容器 */
.input-area {
  display: flex;
  align-items: flex-end;
  gap: 12px;
  padding: 12px 16px;
  background: #FFFFFF;
  border-top: 1px solid #E0E0E0;
}

/* 输入框 */
.input-textarea {
  flex: 1;
  min-height: 40px;
  max-height: 120px;
  padding: 12px 16px;
  border: 1px solid #E0E0E0;
  border-radius: 8px;
  font-size: 14px;
  resize: none;
  overflow-y: auto;
  transition: border-color 0.2s;
}

.input-textarea:focus {
  outline: none;
  border: 2px solid #1F77B4;
}

.input-textarea:disabled {
  opacity: 0.7;
  cursor: not-allowed;
}

.input-textarea::placeholder {
  color: #999999;
}

/* 发送按钮 */
.send-button {
  width: 40px;
  height: 40px;
  border: none;
  border-radius: 8px;
  background: #1F77B4;
  color: #FFFFFF;
  cursor: pointer;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: opacity 0.2s;
}

.send-button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}

.send-button:hover:not(:disabled) {
  opacity: 0.9;
}

/* 字符计数器 */
.char-counter {
  position: absolute;
  bottom: 8px;
  right: 16px;
  font-size: 12px;
  color: #999999;
}

.char-counter.warning {
  color: #FF7F0E;
}

.char-counter.error {
  color: #D62728;
}
```
