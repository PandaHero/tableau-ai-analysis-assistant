## 设置项详细说明

### 1. 数据源选择

| 属性 | 说明 |
|-----|------|
| 类型 | 下拉选择 |
| 数据来源 | Tableau Extensions API 获取当前仪表板的数据源列表 |
| 显示格式 | `数据源名称 (项目名)` |
| 默认值 | 自动选择第一个数据源 |
| 传递方式 | 前端传递数据源名称，后端通过 GraphQL 解析 LUID |

**注意：** Tableau Extensions API 只能获取数据源名称（`DataSource.name`），无法直接获取 LUID。后端负责将名称转换为 LUID 后调用 VizQL API。

### 2. 分析深度

| 属性 | 说明 |
|-----|------|
| 类型 | 下拉选择 |
| 选项 | 标准 / 深入分析 |
| 默认值 | 标准 |

**选项说明：**

| 选项 | 后端值 | 完成度阈值 | 最大轮数 | 说明 |
|-----|-------|----------|---------|------|
| 标准 | `detailed` | 80% | 3 轮 | 快速获取答案，1-2 段落 |
| 深入分析 | `comprehensive` | 95% | 5 轮 | 完整报告，含趋势预测和建议 |

**提示文字：** 显示在下拉框下方，帮助用户理解选项含义

### 3. AI 模型选择

|-----|------|
| 类型 | 下拉选择（分组显示） |
| 默认值 | DeepSeek |

| 设置项 | 存储位置 | 说明 |
|-------|---------|------|
| 主题 | localStorage | 浏览器本地存储 |
| 语言 | localStorage | 浏览器本地存储 |
| 分析深度 | localStorage | 浏览器本地存储 |

## 后端存储实现

使用现有的 `StoreManager` 进行持久化存储：

```python
from tableau_assistant.src.capabilities.storage.store_manager import get_store_manager

store = get_store_manager()

# 保存自定义模型（永久存储，ttl=None）
store.put(
    namespace=("custom_models",), 
    key="my-llama",
    value={
        "name": "My-LLaMA",
        "api_base": "http://localhost:8000/v1",
        "api_key": "sk-xxx",  # 加密存储
        "model_id": "llama-3-70b",
        "created_at": time.time()
    },
    ttl=None
)

# 获取所有自定义模型
models = store.search(namespace_prefix=("custom_models",))

# 删除自定义模型
store.delete(namespace=("custom_models",), key="my-llama")
```

设置变更后立即生效，无需手动保存。
