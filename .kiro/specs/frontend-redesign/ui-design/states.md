# 状态指示器设计

## ThinkingIndicator 思考状态指示器

### 基本结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 🤖 AI 助手                                                              │
│                                                                          │
│   ● ● ●  正在构建查询...                                                │
│                                                                          │
│   ✓ 理解问题                                                            │
│   ● 构建查询  ← 当前阶段                                                │
│   ○ 执行分析                                                            │
│   ○ 生成洞察                                                            │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

### 三点动画

```
帧 1: ●○○
帧 2: ○●○
帧 3: ○○●
帧 4: ○●○
...循环

动画周期: 1.5 秒
```

### 处理阶段

| 阶段 | 状态文字 | 图标 |
|-----|---------|------|
| understanding | 理解问题... | ● |
| building | 构建查询... | ● |
| executing | 执行分析... | ● |
| generating | 生成洞察... | ● |
| error | 处理出错 | ⚠️ |

### 阶段状态图标

| 状态 | 图标 | 颜色 |
|-----|------|------|
| 待执行 | ○ | #E0E0E0 |
| 进行中 | ● | #1F77B4 |
| 已完成 | ✓ | #2CA02C |
| 错误 | ⚠️ | #D62728 |

### 状态切换动画

```
状态文字切换: 300ms 淡入淡出
opacity: 1 → 0 → 1
```

### 错误状态

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 🤖 AI 助手                                                              │
│                                                                          │
│   ⚠️ 处理出错                                                           │
│                                                                          │
│   ✓ 理解问题                                                            │
│   ✓ 构建查询                                                            │
│   ⚠️ 执行分析  ← 错误发生位置                                           │
│   ○ 生成洞察                                                            │
│                                                                          │
│   错误信息: 数据源连接失败                                              │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

- 指示器变为红色 #D62728
- 显示错误消息文字

## ErrorMessage 错误消息

### 基本结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ ⚠️ 网络连接失败，请检查网络后重试                          [重试]      │
└─────────────────────────────────────────────────────────────────────────┘
```

### 错误样式

| 属性 | 值 | 说明 |
|-----|-----|------|
| 背景色 | #FEE2E2 | 浅红背景 |
| 边框 | 1px solid #D62728 | 红色边框 |
| 圆角 | 8px | border-radius |
| 内边距 | 12px 16px | padding |
| 图标 | ⚠️ | 警告图标 |
| 文字色 | #991B1B | 深红色 |

### 错误类型

| 类型 | 消息 | 操作按钮 |
|-----|------|---------|
| 网络错误 | 网络连接失败，请检查网络后重试 | [重试] |
| 请求错误 (400) | 请求格式错误 | - |
| 未授权 (401) | 请重新登录 | [登录] |
| 禁止访问 (403) | 无访问权限 | - |
| 未找到 (404) | 资源不存在 | - |
| 超时 (408) | 分析时间较长，请稍候或尝试简化问题 | [取消] [重试] |
| 服务器错误 (500) | 服务器内部错误，请稍后重试 | [重试] |
| SSE 断连 | 连接已断开 | [重新连接] |

### 内联错误

在消息流中显示的错误：

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 🤖 AI 助手                                                              │
│                                                                          │
│ ┌─────────────────────────────────────────────────────────────────────┐ │
│ │ ⚠️ 分析过程中出现错误                                               │ │
│ │                                                                      │ │
│ │ 数据源连接失败（错误码：DS_001），请联系管理员                       │ │
│ │                                                                      │ │
│ │                                                    [重试] [取消]     │ │
│ └─────────────────────────────────────────────────────────────────────┘ │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## ConnectionStatus 连接状态

### 状态指示器（在 HeaderBar 中）

```
┌─────────────────────────────────────────────────────────────────────────┐
│ [Logo] Tableau AI 助手                              🟢 已连接    [⚙️]  │
└─────────────────────────────────────────────────────────────────────────┘
```

| 状态 | 图标 | 颜色 | 说明 |
|-----|------|------|------|
| 已连接 | 🟢 | #2CA02C | 正常状态 |
| 连接中 | 🟡 | #FF7F0E | 正在连接 |
| 已断开 | 🔴 | #D62728 | 连接断开 |

## CSS 实现

/* 思考指示器 */
.thinking-indicator {
  padding: 16px;
}

.thinking-dots {
  display: flex;
  gap: 4px;

  width: 8px;

  border-radius: 50%;

  animation-delay: 0.5s;
}

  animation-delay: 1s;
}

  0%, 100% { opacity: 0.3; }
  50% { opacity: 1; }

.thinking-status {
  font-size: 14px;

  transition: opacity 0.3s;

}

  50% { opacity: 0.5; }
