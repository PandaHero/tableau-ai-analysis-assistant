# UI 设计文档

本目录包含 Tableau AI 助手前端界面的详细 UI 设计文档。

## 文档结构

| 文件 | 描述 |
|-----|------|
| [layout.md](./layout.md) | 整体布局设计 |
| [chat.md](./chat.md) | 对话界面设计 |
| [data-display.md](./data-display.md) | 数据展示组件设计 |
| [input.md](./input.md) | 输入区域设计 |
| [states.md](./states.md) | 状态指示器设计 |
| [settings.md](./settings.md) | 设置页面设计 |

## 设计原则

1. **Tableau 风格融合**：使用 Tableau 配色方案和设计语言
2. **简洁专业**：专注于数据分析，避免视觉干扰
3. **响应式设计**：支持 320px - 1920px 窗口宽度
4. **可访问性**：符合 WCAG 2.1 AA 标准

## 配色方案

### Tableau 10 色板

| 颜色 | 色值 | 用途 |
|-----|------|------|
| 主蓝 | #1F77B4 | 品牌色、主要按钮、用户消息 |
| 橙色 | #FF7F0E | 警告、异常洞察 |
| 绿色 | #2CA02C | 成功、建议洞察 |
| 红色 | #D62728 | 错误、负数 |
| 紫色 | #9467BD | 图表配色 |
| 棕色 | #8C564B | 图表配色 |
| 粉色 | #E377C2 | 图表配色 |
| 灰色 | #7F7F7F | 次要文字、边框 |
| 黄绿 | #BCBD22 | 图表配色 |
| 青色 | #17BECF | 图表配色 |

### 中性色

| 颜色 | 色值 | 用途 |
|-----|------|------|
| 白色 | #FFFFFF | 背景、AI 消息背景 |
| 浅灰 | #F5F5F5 | 芯片背景、悬停状态 |
| 边框灰 | #E0E0E0 | 边框、分隔线 |
| 文字灰 | #666666 | 次要文字 |
| 深灰 | #333333 | 主要文字 |

## 响应式断点

| 断点 | 宽度范围 | 布局模式 |
|-----|---------|---------|
| minimal | 320px - 479px | 最小化布局 |
| compact | 480px - 767px | 紧凑布局 |
| standard | ≥ 768px | 标准布局 |
