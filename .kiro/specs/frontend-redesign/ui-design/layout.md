# 整体布局设计

## 三区域布局

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        HeaderBar (48px)                                  │
│                                                              ┌────────┐ │
│                    [Logo] Tableau AI 助手                    │ ⚙️设置 │ │
│                         (居中显示)                           │ 24x24  │ │
│                                                              └────────┘ │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│                                                                          │
│                                                                          │
│                     ChatContainer (flex: 1)                              │
│                        可滚动区域                                        │
│                                                                          │
│                                                                          │
│                                                                          │
│                                                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                        InputArea (64px)                                  │
│  ┌─────────────────────────────────────────────────────────────┐ ┌───┐ │
│  │ 请输入您的问题...                                            │ │ ➤ │ │
│  └─────────────────────────────────────────────────────────────┘ └───┘ │
└─────────────────────────────────────────────────────────────────────────┘
```

## 尺寸规范

### HeaderBar 顶部导航栏

| 属性 | 值 | 说明 |
|-----|-----|------|
| 高度 | 48px ± 2px | 固定高度 |
| 内边距 | 0 16px | 左右内边距 |
| 背景色 | #FFFFFF | 白色背景 |
| 边框 | 1px solid #E0E0E0 | 底部边框 |
| Logo 尺寸 | 32x32px | Tableau 风格 Logo |
| 标题字号 | 16px | font-weight: 600 |
| 布局 | 居中 | Logo + 标题居中显示 |
| 设置按钮 | 24x24px | 右侧绝对定位 |

### ChatContainer 对话区域

| 属性 | 值 | 说明 |
|-----|-----|------|
| 高度 | flex: 1 | 占据剩余空间 |
| 内边距 | 16px | 四周内边距 |
| 背景色 | #FAFAFA | 浅灰背景 |
| 滚动 | overflow-y: auto | 垂直滚动 |
| 消息间距 | 16px | 消息之间的间距 |

### InputArea 输入区域

| 属性 | 值 | 说明 |
|-----|-----|------|
| 高度 | 64px ± 2px | 固定高度 |
| 内边距 | 12px 16px | 内边距 |
| 背景色 | #FFFFFF | 白色背景 |
| 边框 | 1px solid #E0E0E0 | 顶部边框 |
| 输入框高度 | 40px - 120px | 自动扩展 |
| 发送按钮 | 40x40px | 圆形按钮 |

## 响应式布局

### 标准布局 (≥ 768px)

```
┌─────────────────────────────────────────────────────────────────────────┐
│ [Logo] Tableau AI 助手                                           [⚙️]  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│                        对话区域                                          │
│                      (间距: 16px)                                        │
│                                                                          │
├─────────────────────────────────────────────────────────────────────────┤
│ [输入框...                                                   ] [发送▶]  │
└─────────────────────────────────────────────────────────────────────────┘
```

- 完整显示标题文字
- 标准间距 16px
- 消息最大宽度 80%/90%

### 紧凑布局 (480px - 767px)

```
┌─────────────────────────────────────────────────────────────┐
│ [Logo]                                                [⚙️]  │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│                      对话区域                                │
│                    (间距: 12px)                              │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│ [输入框...                                       ] [▶]      │
└─────────────────────────────────────────────────────────────┘
```

- 隐藏标题文字
- 缩小间距至 12px
- 消息最大宽度 85%/95%

### 最小化布局 (320px - 479px)

```
┌─────────────────────────────────────────┐
│ [Logo]                            [⚙️]  │
├─────────────────────────────────────────┤
│                                          │
│              对话区域                    │
│            (间距: 8px)                   │
│                                          │
├─────────────────────────────────────────┤
│ [输入框...                   ] [▶]      │
└─────────────────────────────────────────┘
```

- 仅显示 Logo 和必要按钮
- 最小间距 8px
- 消息最大宽度 90%/95%

### 窗口过小 (< 320px)

```
┌─────────────────────────────┐
│                              │
│      ⚠️ 窗口过小            │
│                              │
│   请调整窗口大小以获得      │
│   最佳体验                   │
│                              │
└─────────────────────────────┘
```

## CSS 实现

```css
/* 布局容器 */
.layout-container {
  display: flex;
  flex-direction: column;
  height: 100vh;
  width: 100%;
  min-width: 320px;
  max-width: 1920px;
  margin: 0 auto;
}

/* 顶部导航栏 */
.header-bar {
  height: 48px;
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 0 16px;
  background: #FFFFFF;
  border-bottom: 1px solid #E0E0E0;
  flex-shrink: 0;
}

/* 对话区域 */
.chat-container {
  flex: 1;
  overflow-y: auto;
  padding: 16px;
  background: #FAFAFA;
}

/* 输入区域 */
.input-area {
  height: 64px;
  display: flex;
  align-items: center;
  gap: 12px;
  padding: 12px 16px;
  background: #FFFFFF;
  border-top: 1px solid #E0E0E0;
  flex-shrink: 0;
}

/* 响应式断点 */
@media (max-width: 767px) {
  .header-bar .title {
    display: none;
  }
  .chat-container {
    padding: 12px;
  }
  .input-area {
    padding: 12px;
  }
}

@media (max-width: 479px) {
  .chat-container {
    padding: 8px;
  }
  .input-area {
    padding: 8px;
  }
}
```
