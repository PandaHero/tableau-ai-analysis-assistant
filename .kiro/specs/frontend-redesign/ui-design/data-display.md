# 数据展示组件设计

## DataTable 数据表格

### 基本结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 📊 查询结果                                              [导出 CSV]    │
├─────────────────────────────────────────────────────────────────────────┤
│ ┌─────────┬─────────────┬─────────┬─────────┬───────────────────────┐  │
│ │ 区域 ▲  │ 销售额      │ 订单数  │ 客单价  │ 同比增长              │  │
│ ├─────────┼─────────────┼─────────┼─────────┼───────────────────────┤  │
│ │ 华东    │ ¥456,789    │ 1,234   │ 370.21  │ +23.5%                │  │
│ │ 华南    │ ¥345,678    │ 987     │ 350.18  │ +15.2%                │  │
│ │ 华北    │ ¥234,567    │ 756     │ 310.27  │ +8.7%                 │  │
│ │ 华中    │ ¥198,765    │ 654     │ 303.92  │ +5.3%                 │  │
│ │ 西南    │ ¥156,432    │ 543     │ 288.09  │ -2.1%                 │  │
│ │ ...     │ ...         │ ...     │ ...     │ ...                   │  │
│ └─────────┴─────────────┴─────────┴─────────┴───────────────────────┘  │
│                                                                          │
│ ┌─────────────────────────────────────────────────────────────────────┐ │
│ │                    展开全部（共 15 行）                              │ │
│ └─────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────┘
```

### 表格规格

| 属性 | 值 | 说明 |
|-----|-----|------|
| 默认显示行数 | 10 行 | 超过时显示展开按钮 |
| 表头背景 | #F5F5F5 | 浅灰背景 |
| 表头字重 | 600 | font-weight |
| 行高 | 40px | line-height |
| 单元格内边距 | 8px 12px | padding |
| 边框 | 1px solid #E0E0E0 | 单元格边框 |
| 悬停行背景 | #F9F9F9 | hover 状态 |

### 排序功能

| 状态 | 图标 | 说明 |
|-----|------|------|
| 未排序 | - | 无图标 |
| 升序 | ▲ | 首次点击 |
| 降序 | ▼ | 再次点击 |
| 取消排序 | - | 第三次点击 |

### 数值格式化

| 类型 | 格式 | 示例 |
|-----|------|------|
| 整数 | 千分位分隔 | 1,234,567 |
| 小数 | 保留 2 位 | 3.14 |
| 负数 | 红色显示 | -2.1% |
| 百分比 | 带 % 符号 | 23.5% |
| 货币 | 带 ¥ 前缀 | ¥456,789 |

### 长文本处理

**方案：自动换行 + 可展开**

对于长文本单元格，采用以下策略：

```
默认状态（最多显示 3 行）：
┌─────────────────────────────────────────────────────────────────────────┐
│ 这是一段很长的文本内容，可能包含详细的描述信息。当文本超过三行时，      │
│ 会自动截断并显示"展开"按钮。用户可以点击展开查看完整内容...            │
│ [展开 ▼]                                                                │
└─────────────────────────────────────────────────────────────────────────┘

展开状态：
┌─────────────────────────────────────────────────────────────────────────┐
│ 这是一段很长的文本内容，可能包含详细的描述信息。当文本超过三行时，      │
│ 会自动截断并显示"展开"按钮。用户可以点击展开查看完整内容。这里是        │
│ 完整的文本内容，包括所有的详细信息和说明。用户可以看到全部内容，        │
│ 然后点击"收起"按钮将其折叠回去。                                        │
│ [收起 ▲]                                                                │
└─────────────────────────────────────────────────────────────────────────┘
```

**规则：**
- 默认显示最多 3 行（约 150 字符）
- 超过 3 行时显示"展开 ▼"按钮
- 点击展开显示完整内容 + "收起 ▲"按钮
- 文本自动换行（word-wrap: break-word）

└─────────────────────────────────────────────────────────────────────────┘
```

## SuggestedQuestions 推荐问题

### 基本结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 💬 您可能还想了解                                                       │
│                                                                          │
│ ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐          │
│ │ 电子产品细分分析 │ │ 销售趋势对比     │ │ 更多 ▼           │          │
│ └──────────────────┘ └──────────────────┘ └──────────────────┘          │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

### 芯片样式

| 状态 | 背景色 | 边框色 | 说明 |
|-----|-------|-------|------|

```
